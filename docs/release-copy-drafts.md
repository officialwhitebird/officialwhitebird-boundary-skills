## Release Note

`officialwhitebird-boundary-skills` packages four local boundary checks into one agent-readable workflow:

- `handoff-lint` for work instruction structure.
- `policy-lint` for proposed command classification.
- `contract-lint` for declared contract vs recorded result checks.
- `claim-lint` for public-copy and internal-name scans.

The repository includes generic fixtures, a local PowerShell case-study runner, and a visual case-study page that shows the four boundaries together.
